* 赛道/概念名称：锂电赛道
* 研究时间范围：2025-04-15至2026-04-15
* 核心结论：过去12个月，锂电赛道并未被证伪，需求、出口和储能仍在增长，但“整条产业链一起受益”的老主线已经不成立。最新数据更像是“总量增长、盈利分化、主线切到分支”：储能与出口明显强于国内动力装车，碳酸锂价格从2025年1-4月均价约7.4万元/吨，升至2026年4月14日SMM电池级碳酸锂均价16.15万元/吨；与此同时，监管层在2026年1月和4月连续部署整治动力及储能电池行业“内卷式”竞争，说明行业的核心矛盾已从“有没有需求”转向“有没有利润、有没有秩序”。因此，当前不宜把“广义锂电”当成单一大主线，更值得跟踪的是储能出口、锂价修复、磷酸铁锂材料盈利修复，以及固态/高压快充这类技术分支。([wap.miit.gov.cn][1])
* 当前状态（强化 / 维持 / 弱化 / 证伪 / 主线切换）：主线切换
* 当前主要驱动：多因素共振，以需求驱动、价格驱动、政策驱动为主
* 当前更像（早期线索 / 正在形成主线 / 已成主线 / 高位拥挤 / 退潮旧题材）：退潮旧题材中的分支重构

重要事件时间线：

1. 时间：2025-06-13

   * 事件：工信部披露2025年1-4月全国锂电池总产量超473GWh，同比增长68%；其中储能型锂电池产量超110GWh，出口总额1554亿元，同比增长25%。
   * 性质（新增信息 / 重复确认 / 强化 / 弱化 / 证伪）：新增信息、强化
   * 对赛道影响：说明锂电需求并未塌陷，且储能和出口比单纯国内车用需求更强，是后续主线切换的起点。([wap.miit.gov.cn][1])

2. 时间：2025-09-12

   * 事件：国家发展改革委、国家能源局印发《新型储能规模化建设专项行动方案（2025—2027年）》。
   * 性质：新增信息、强化
   * 对赛道影响：政策把储能从“景气分支”进一步推向“制度支持分支”，强化了锂电赛道内“储能优先于整链普涨”的逻辑。([NDRC][2])

3. 时间：2025-11至2025-12

   * 事件：中国有色金属工业协会锂业分会披露，2025年11月电池级碳酸锂价格从月初约8.1万元/吨升至月末约9.2万元/吨，期货价格一度突破10万元/吨。
   * 性质：新增信息、强化
   * 对赛道影响：锂价由长期下行转向修复，上游锂资源、锂盐和LFP材料环节开始出现利润修复预期，市场关注点从“量增”转向“价利修复”。([QQ News][3])

4. 时间：2026-01-07至2026-01-14

   * 事件：四部门召开动力和储能电池行业座谈会，明确行业存在盲目建设、低价竞争等非理性行为；同期，2025年中国新能源汽车销量1649万辆，同比增长28.2%，新能源汽车出口261.5万辆，同比增长103.7%。
   * 性质：重复确认、强化、弱化并存
   * 对赛道影响：需求端继续增长，说明赛道没有被证伪；但政策层明确“内卷”问题，说明行业利润修复不能单靠销量增长，广义锂电进入“有量无价”向“有量有序”过渡阶段。([miit.gov.cn][4])

5. 时间：2026-01-19至2026-01-30

   * 事件：湖南裕能预计2025年净利润同比增长93.75%至135.87%；赣锋锂业预计2025年扭亏为盈；天齐锂业预计2025年扭亏为盈，称锂精矿定价周期错配影响大幅减弱、成本逐步贴近最新采购价格。
   * 性质：新增信息、强化
   * 对赛道影响：这是过去12个月里最重要的一组“利润修复”证据，说明市场若继续炒锂电，更可能从整车销量映射，切换到上游/材料盈利拐点映射。([Sina Finance][5])

6. 时间：2026-04-09至2026-04-10

   * 事件：四部门再次召开动力及储能电池行业企业座谈会，要求推进产能预警调控、规范价格竞争；同月数据表明，2026年1-3月动力和储能电池累计产量487.4GWh，同比增长49.3%，储能电池累计销量145.1GWh，同比增长111.8%，累计出口84.1GWh，同比增长36.7%，但国内动力电池累计装车量124.9GWh，同比下降4.1%；同期新能源汽车出口95.4万辆，同比增长1.2倍。
   * 性质：新增信息、重复确认、主线切换
   * 对赛道影响：这是当前最关键的结构证据。赛道总量继续扩张，但真正强的是储能、出口和部分价格弹性；国内动力装车开始显示“增速换挡”，广义锂电已从旧主线切向分支线。([wap.miit.gov.cn][6])

赛道动态分析：

* 强化逻辑的证据：2025年1-4月锂电总产量和出口高增，2025全年新能源汽车销量和出口再创新高，2026年一季度储能电池销量同比翻倍、出口继续增长，说明赛道需求端仍在扩张，且储能/出海是更强增量。锂价从2025年低位回升，也让上游与材料利润修复逻辑成立。([wap.miit.gov.cn][1])
* 削弱逻辑的证据：2026年一季度国内动力电池装车量同比下降4.1%，2026年一季度新能源汽车销量同比下降3.7%；监管层连续两次部署治理低价竞争，说明行业仍处于产能和价格竞争压力之下，广义锂电暂不足以形成“整链持续主线”。([Sina Finance][7])
* 是否出现分支逻辑：出现了。过去市场更多围绕“新能源车放量=锂电普涨”，现在更像三条分支：一是储能出口，二是锂价修复带来的上游/材料利润弹性，三是固态电池、高压快充这类技术分支。固态电池仍处产业化前期，欧阳明高近期仍提示全固态规模化量产尚待3—5年，因此它更偏主题分支，不是当下业绩主线。([NDRC][2])
* 当前市场更可能炒什么：更可能炒分支逻辑，其次是情绪映射。最容易形成交易闭环的是“储能出口+锂价修复”，而不是“整条锂电链一起上涨”。
* 当前研究价值判断：有研究价值，但不建议再把“锂电”作为一个笼统主题跟踪；更适合拆成电芯龙头、LFP材料、锂资源价格弹性、技术分支四条线，研究优先级中等。

A股映射方向：

* 直接受益方向：动力+储能双轮驱动的电芯龙头；磷酸铁锂正极材料龙头；在锂价回升时成本/售价联动更快的锂盐和锂资源龙头。([CATL][8])
* 间接受益方向：电解液、隔膜、结构件等材料与零部件龙头，受益于总量增长，但盈利修复通常滞后于电芯/上游。([Cninfo][9])
* 概念映射方向：锂电设备、固态电池设备与材料、回收利用等，往往受技术主题和政策主题驱动，基本面兑现节奏慢于市场交易节奏。([Shenzhen Stock Exchange][10])

   * 归类理由：动力+储能双龙头，最符合当前“从动力扩散到储能/出海”的真实主线。公司2024年动力电池系统收入2530.41亿元、储能电池系统收入572.90亿元，境外收入占比30.48%；公司年报披露其2024年动力电池全球市占率37.9%、储能电池全球市占率36.5%。([CATL][8])

     * 主题证据强度：19
     * 受益直接性：18
     * 业绩弹性：17
     * 业务匹配度与映射确定性：19
     * 催化落地确定性与持续性：17
     * 逻辑相关度总分：90
   * 市场主炒概率：

     * 判断结果：中

     * 叙事简洁度依据：“储能增长+全球龙头”叙事清晰。
     * 板块代表性依据：对锂电板块具有最强代表性。
     * 注意力与情绪承载依据：辨识度高，但大市值决定其更偏中军而非短线情绪龙。

   * 市场主炒概率：

     * 题材辨识度依据：储能、电池出海、商用车电池三重辨识度。

     * 板块代表性依据：材料端代表性较强。
     * 注意力与情绪承载依据：弹性好于电芯中军，但板块号召力弱于上游锂资源。
     * 使用的证据类型：年报、业绩预告、行业结构数据

   * 主要不确定性：行业竞争激烈，升级逻辑兑现速度不确定。

2. 先导智能

   * 归类理由：与锂电长期高度相关，但当前更偏“设备预期股”而非赛道即期受益股。公司年报明确提到，2024年全球新能源市场仍增长，但受行业产能短期供求错配影响，锂电设备行业承压；公司2024年营收同比下降28.71%。这意味着它与锂电景气相关，但本轮盈利兑现明显滞后于电芯、上游和部分材料。([Shenzhen Stock Exchange][10])
   * 逻辑相关度评分：

     * 主题证据强度：12
     * 受益直接性：10
     * 业绩弹性：11
     * 业务匹配度与映射确定性：13
     * 催化落地确定性与持续性：9
     * 逻辑相关度总分：55
   * 市场主炒概率：

     * 判断结果：低
     * 题材辨识度依据：设备龙头辨识度高，但并非当前最顺周期位置。
     * 叙事简洁度依据：需要解释“订单—交付—收入”链条，叙事不够短。
     * 板块代表性依据：设备端代表性强。
     * 注意力与情绪承载依据：适合中长期景气修复交易，不适合当前主线短炒。
     * 使用的证据类型：年报、行业景气表述、营收数据
     * 置信度：中
   * 证伪清晰度：18/20
   * 证伪条件：若后续2-3个季度新签订单和收入未出现改善，且下游扩产继续收缩。
   * 证伪条件分类：基本面证伪、产业链证伪、时间维度证伪
   * 证伪后处理建议：直接剔除
   * 主要不确定性：下游产能利用率和新增CAPEX恢复节奏。

最终动态判断：

* 当前主线是否仍成立：广义锂电“整链共振主线”不再成立，但赛道基本面没有被证伪；当前成立的是分支主线。
* 当前更值得跟踪主线还是分支线：分支逻辑
* 当前研究优先级（高 / 中 / 低）：中
* 我后续最应该跟踪的3个变量：

  1. 电池级碳酸锂价格能否稳定在10万元/吨以上并带来材料/上游持续盈利修复。([hq.smm.cn][15])
  2. 储能电池销量与出口能否继续显著跑赢动力电池装车。([NE时代][16])
  3. 反“内卷”政策是否真正落到价格竞争、账期和产能预警上。([miit.gov.cn][4])
* 总结：锂电赛道现在最重要的不是“有没有需求”，而是“需求增量落在哪一段、利润修复落在哪一段”。我的结论不是看空锂电，而是反对再把它当成一个笼统题材。当前更值得盯的是储能出口、电芯龙头、LFP材料盈利修复，以及锂价修复带来的上游情绪锚；至于固态、高压快充，更适合作为分支主题跟踪，而不是直接当成当期业绩主线。
