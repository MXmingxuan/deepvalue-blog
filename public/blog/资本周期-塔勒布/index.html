从脆弱到反脆弱，资本周期视角下的投资思考

2025/12/1
交易

在英国马拉松资本的资本周期框架里，有一个很朴素的出发点：
别太迷信需求预测，多花时间盯供给。

在塔勒布的《反脆弱》里，把世界分成三类：脆弱（玻璃杯一摔就碎）、强韧（怎么摔都不碎）、反脆弱（越摔越坚韧）。

如果把这个关于脆弱的框架放在资本周期方法上看，会发现：

  • 需求预测这件事，本身是脆弱的；

  • 供给端分析本身偏强韧；

  • 在供给端视角上设计出来的资本周期策略，有机会变成反脆弱。

下面分几层展开分析。

01 需求预测为什么天然脆弱

当你试图预测需求时，你面对的是一个“复杂适应系统”：

  • 宏观政策的情绪变化；

  • 技术的突变（AI ）；

  • 人类偏好的突然转向；

  • 黑天鹅事件：疫情、战争、制裁、监管收紧等等。

这些变量叠在一起，符合塔勒布说的“极端斯坦”特征：
尾部很粗，少数事件决定大部分结果，而且事先很难穷举。

现实中的需求预测模型，往往有几个隐含假设：

  1. 变量大致服从正态分布；

  2. 历史可以线性外推；

  3. 大部分时间是“小波动 + 偶尔修正”。

一旦来一只真正的黑天鹅，比如 2008 年金融危机、2020 年疫情，
这些模型不是“有一点误差”，而是整体失效，所有 Excel 里的格子要重算。

脆弱的典型特征，就是对大冲击没有缓冲区，一碰就碎。
需求预测正是如此：

  • 你越依赖一个具体数字（比如“明年行业需求增长 8%”），

  • 一旦现实偏离这个点，整个投资逻辑就跟着摇晃。

02 供给端：受物理和时间约束的“中庸国”

马拉松资本的资本周期分析，本质上是从供给端理解行业：
高回报 → 吸引资本 → 扩张产能 → 供给过剩 → 回报下降 → 资本撤出 → 供给收缩 → 回报回升。

相对需求端，供给有几个重要特征。

1. 物理约束：不是说改就能改

建一座铜矿，需要多年勘探、审批、建设；
造一艘大船，需要好几年；
建一条大型炼化、一个晶圆厂，也要长周期的投入。

这类供给，是被物理现实“锁住”的。
不会因为央行行长讲了几句话，产能就凭空多 20%。

所以，哪怕宏观预期晃来晃去，
未来几年大致会有哪些项目投产、有哪些产能退出，是可以被列出来、一个个数清楚的。

2. 可观察性：供给是存量，需求是流量

供给，大多以存量的形式存在：

  • 现有产能（矿山、船队、工厂）；

  • 已经立项、在建的项目；

  • 已经宣布但尚未开工的资本开支计划。

这些东西可以被统计、被跟踪，甚至通过年报、招股书、行业数据大致拼出来

需求则不同：

  • 它是一个流量，是人们的偏好、预算、情绪的综合结果；

  • 很多时候，需求的“拐点”只会事后才被确认。

你可以数清楚港口里有多少艘船、全世界有多少万吨产能，
但你数不清，明年有多少人突然决定少用一点铜、多出几趟远洋。

3. 滞后性：今天的 Capex，决定明天的供给

供给端变化的一个关键特点，是滞后性很强：

  • 今天公司做出的资本开支决策（Capex），

  • 会在几年后变成新增产能，

  • 再过几年才会完全反映到行业利润率上。

这意味着：

  • 只要你认真盯着行业 Capex、项目进度、资产负债表，

  • 你对中期供给的判断，其实是一种“已知的已知”，

  • 不需要对未来做太多想象，只要照着这些硬数据推就行。

03 塔勒布的三类世界：脆弱、强韧、反脆弱

塔勒布的三分法，可以简单记成：

  • 脆弱：讨厌波动，遇到冲击就坏得更快（瓷杯、加杠杆的资产）。

  • 强韧：能扛住波动，但自身变化不大（石头、防守性资产）。

  • 反脆弱：需要波动，靠冲击变得更好（肌肉、期权、多样化的小赌注）。

这套框架的关键，是看一个系统面对不确定性时，
结果是“凹的”还是“凸的”：

  • 凹：坏事来的时候，下跌空间远大于上涨空间 → 脆弱；

  • 平：好坏都有限 → 强韧；

  • 凸：坏事有限，但一旦好事来，收益向上张得很开 → 反脆弱。

接下来，我们可以把资本周期里的几层东西放进这个框架里看。

04 第一层：从需求预测转向供给观察，本身就是走向强韧

仅仅是把注意力从需求预测，转向供给端分析，本身就是一种“强韧化”。

大多数投资者在问：

“明年需求会不会增长 8%？”
“这条赛道能不能保持 20% 复合增速？”

而资本周期投资者在问：

“这个行业过去几年赚了这么多钱，现在有多少资本冲进来了？”
“有没有出现过度扩产？有没有疯狂并购和 IPO？”

这两种问法的差别在于：

  • 前者直接站在“极端斯坦”的需求世界里，去猜一个本质上难预测的数字；

  • 后者站在“中庸国”的供给世界里，去看那些已经发生、正在发生的资本流动。

这样做有几个好处：

  1. 减少对单点预测的依赖
    你不再需要押注一个精确的需求增速，只要判断：

    • 行业是不是已经“被资本宠坏了”；

    • 还是已经“被资本抛弃了”。

  2. 对噪音更不敏感
    短期的经济数据、情绪变化，
    对“未来几年会有多少船交付、多少矿山投产”影响有限。
    你的思考周期被自然拉长。

  3. 决策更靠近常识
    高回报吸引资本、过度扩张导致回报下降、资本撤出后留下好机会，
    这些是跨行业、跨时代都成立的朴素规律。

从塔勒布的角度看，
这种从“预测需求”转向“观察供给”的改变，至少可以把系统从脆弱拉到强韧。
你不一定从波动中获益，但你更不容易被一次需求判断失误打垮。

05 第二层：资本周期策略如何变成“反脆弱”

更有意思的一层，是当你用供给端分析去构建具体策略时，
有机会把自己放在一个“凸性”的位置上，让组合变成反脆弱。

1. 行业崩盘时：捡起被丢在地上的“期权”

典型的资本周期路径是这样的：

  1. 行业高景气 → 利润率很高 → 大量资本涌入 → 疯狂扩产；

  2. 供给上来后，需求不及预期 → 价格和利润率下行；

  3. 资本开始撤离 → 公司破产、并购、减记、停工，行业进入“至暗时刻”；

  4. 供给被动收缩 + 未来 Capex 枯竭 → 为下一轮盈利修复埋下伏笔。

在第三阶段，整个行业看起来一片狼藉：

  • 股价跌去 70%、80%；

  • 新闻全是负面；

  • 卖方报告集体下调评级。

但如果你站在供给端视角，会发现一个“隐藏的免费期权”：

  • 只要需求不进一步崩掉，哪怕只是“从很差恢复到一般”**；

  • 在供给被严重伤害、没有新增资本敢进来的情况下，

  • 幸存者的利润弹性会很夸张。

如果这时你买入的是：

  • 管理层克制、不乱扩张；

  • 资产负债表相对健康；

  • 成本曲线靠前的那一批幸存者

那你的头寸就有点像一个期权：

  • 最坏情况：行业继续恶化，你的损失被估值压缩和仓位控制所“封顶”；

  • 好的情况：一旦需求稍微恢复，加上供给紧缺，利润和估值双击，上涨空间是“非线性”的。

这就是塔勒布说的“凸性”——坏事有限，好事敞口很大。

2. 从“怕波动”变成“等波动来”

如果你的研究框架是需求预测，你往往害怕波动：

  • 每次宏观数据不及预期，你都要紧张地调模型；

  • 每次政策变化，你都要重写一次行业假设。

但如果你用资本周期视角：

  • 行业越是经历剧烈的“破坏”（亏损、破产、削减 Capex），

  • 未来某个时间点出现“反转”的概率就越高。

这时，你反而会期待：

“再恶劣一点，让更多劣质产能退场，让更多资本撤走。”

系统开始从“怕冲击”变成“靠冲击清理过度资本”，这就是反脆弱的特征。

06

把马拉松资本的资本周期方法，和塔勒布的反脆弱框架放在一起看，可以得到一个简单的归纳：

  1. 需求预测是脆弱的

    • 它高度依赖对复杂系统的单点预测；

    • 遇到黑天鹅就会整体失效。

  2. 供给端分析是强韧的

    • 它基于物理约束、时间滞后和可观察的存量；

    • 不需要特别精确的预测，只需要常识性的判断。

  3. 在供给端视角之上构建的资本周期策略，有机会成为反脆弱

    • 在行业最痛苦、最混乱的时候买入幸存者；

    • 让自己站在一个“下行有限、上行敞口很大”的位置；

    • 借助波动和危机，获得超额收益。
